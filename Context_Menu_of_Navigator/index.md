## Context Menu of Navigator


The main commands of the report server are duplicated in the context menu. This menu will appear when you right-click a mouse (or holding your finger on mobile devices or other alternative action) in the toolbox. Below is the context menu:


![](../images/topics/Context_Menu_of_Navigator_1.png)


As can be seen from the picture above, the top part of the menu is represented with the commands to create basic items:

![](../images/img_1.png) The command invokes a menu to create a report or upload it from the file.

![](../images/img_2.png) The command adds a new data source.

![](../images/img_3.png) The command is use to schedule commands to be executed.

![](../images/img_4.png) The commands creates folders in the list of items. They are needed to organize and store items and other folders. Using to create a hierarchy of folders in the list of items of the report server.


Commands to create items are placed in a separate section:

![](../images/img_5.png) The command is used to invoke the **Calendar**. In the calendar, for example, you can create a schedule to run the scheduler.

![](../images/img_6.png) The command is used to add a file to the list of items in the report server.

![](../images/img_7.png) The command is used to call the menu to create a list of contacts. For example, the contact list may be used by the scheduler to send the result by email.


![](../images/img_8.png) The command is present only in the context menu of an item Report. This command runs a report for rendering and opens the tab view reports.

![](../images/img_9.png) This command calls the edit menu of the selected item.

![](../images/img_10.png) The command deletes selected item.

![](../images/img_11.png) The command is used to call the report designer and is present only in the context menu of an item Report.

![](../images/img_12.png) The command is used to cut items to the clipboard.

![](../images/img_13.png) The command is used to copy items to the clipboard.

![](../images/img_14.png) The command is used to paste items from the clipboard.


Also you should consider that in the context menu of the **Scheduler** the commands to control the scheduler will be present instead of the command Run Report:


![](../images/topics/Context_Menu_of_Navigator_2.png)


![](../images/img_1.png) This command starts the scheduler at once not by a schedule.

![](../images/img_2.png) Using this command the scheduler is running and it is executed according to the schedule.

![](../images/img_3.png) Using this command the scheduler stops.
