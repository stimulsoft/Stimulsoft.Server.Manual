## Report with Label


Creating a report with label using the wizard includes 12 steps. Not all of them are mandatory.


![](../../images/topics/Menu_Create.Report.Label_Report_Wizard_1.png)


![](../../images/img_1.png) Attach other items to the report. For example, images, files, data sources, etc. This is a mandatory step.

![](../../images/img_2.png) Select the data source on which a report will be created. This is a mandatory step.

![](../../images/img_3.png) This step defines the label settings.

![](../../images/img_4.png) Give the name to the report item and write description for the report, if necessary. This is a mandatory step.


![](../../images/topics/Menu_Create.Report.Label_Report_Wizard_2.png)


![](../../images/img_1.png) **Type Panel**. Set the report units and Label Type.

![](../../images/img_2.png) **Panel size label**. Change the size of the label.

![](../../images/img_3.png) **Panel size pages**. Select the page size or manually set the value of the width and height of the page, as well as its borders.

![](../../images/img_4.png) **Panel configuration label**. Customize the label, specify the number of rows and columns, and select the direction of placing labels on the page.

![](../../images/img_5.png) **Preview Panel**. Preview the labels on the page, certain parameters.
