## Report with Chart


Creating a report with chart using the wizard includes 8 steps. Not all of them are mandatory.


![](../../images/topics/Menu_Create.Report.Chart_Wizard_1.png)


![](../../images/img_1.png) Attach other items to the report. For example, images, files, data sources, etc. This is a mandatory step.

![](../../images/img_2.png) Select the chart type.

![](../../images/img_3.png) Select the chart style.

![](../../images/img_4.png) Add the chart series. The chart should be at least one series.

![](../../images/img_5.png) Define a number of labels by selecting them and changing the type of parameters.

![](../../images/img_6.png) Determine the parameters of the axes in the chart.

![](../../images/img_7.png) Set the legend and change its parameters implemented at this stage.

![](../../images/img_8.png) Give the name to the report item and write description for the report, if necessary. This is a mandatory step.
