## Tabs

The tabs are used to display the items of a particular type. Also, the [Users](Users/index.md) and [System](System.md) tabs contain system commands to control users and server system, respectively:


![](../images/tabs.png)


![](../images/img_1.png) The expand/collapse button to the tab panel.

![](../images/img_2.png) The [New tab](New.md). Selecting this tab provides the ability to create a report.

![](../images/img_3.png) The [All Elements tab](All_Elements.md). When you select this tab, you will see all the elements, including folders.

![](../images/img_4.png) The [Reports tab](Reports.md). When you select this tab, folders and report items only will be displayed.
![](../images/img_5.png) The [Dashboards tab](Dashboards.md). Selecting this tab provides the ability to create dashboards.

![](../images/img_6.png) The [Data tab](Data.md). When you select this tab, folders, data sources, and only the files that contain the data (XML, Excel, CSV, DBF, and JSON) will be displayed.

![](../images/img_7.png) The [Schedulers tab](Schedulers.md). When you select this tab, only folders and schedulers will be displayed.

![](../images/img_8.png) The [Users tab](Users/index.md). This tab will display workspaces, roles, and user accounts.

![](../images/img_9.png) The [System tab](System.md). License and Email Templates commands will be displayed.

![](../images/img_10.png) The command is used to call the [About](About.md) menu.

![](../images/img_11.png) The command to navigate the **Stimulsoft Server**. The help files will be opened in a new browser tab.
