## Display panel

The display panel includes commands to control modes of displaying server items:


![](../images/display_panel.png)


![](../images/img_1.png) The full path to the current directory, the folder in which the user is located.

![](../images/img_2.png) The button to enable the List mode. The items will be displayed a list of items with information about their type and date of creation.


![](../images/list.png)


![](../images/img_3.png) The button to enable the **Grid** mode. The list of items will look like big icons as a grid.


![](../images/grid.png)


![](../images/img_4.png) The button is used to enable the **Details** panel. If you click this button, it will display an additional panel with detailed information of items.


![](../images/details.png)


> **Information**
>
> On the **Details** panel, you can change the **Name** and **Description** of the selected item. To do this, click on the name or description and change the text.


![](../images/img_5.png) The button to sort items. It contains the drop-down list, where you can define the type and direction of sorting.


![](../images/drop_down_list.png)


![](../images/img_6.png) The button to enable the [Recycle Bin](Recycle_Bin.md) mode. If the button is enabled, it will display the items of the bin.


![](../images/recycle_bin.png)
