## Toolbar


The main toolbar locates commands to control the report. Below is the structure of the toolbar with the description of each command:


![](../images/topics/Viewer.Toolbar_1.png)


![](../images/img_1.png) Prints a report. After activation of this command the printing dialog with parameters of printing will be displayed.

![](../images/img_2.png) [Saves the rendered report](Exports/index.md) to other file formats.

![](../images/img_3.png) Opens the dialog for changing basic parameters of the rendered report.

![](../images/img_4.png) Shows/hides the tree of bookmarks. If there are no bookmarks in the rendered report then the viewer will automatically hide the tree of bookmarks. If there are bookmarks in a report, then the viewer will automatically show the tree of bookmarks.

![](../images/img_5.png) Enable the search panel.

* **Status Bar**


On the picture below you can see the toolbar that is used for report navigation


![](../images/topics/Viewer.Toolbar_2.png)


![](../images/img_1.png) Set the first page of a report as the current page.

![](../images/img_2.png) Set the previous page of a report as the current one.

![](../images/img_3.png) Show the number of the current page and the number of pages in a report.

![](../images/img_4.png) Set the next page of a report as the current one.

![](../images/img_5.png) Set the last page of a report as the current page.

![](../images/img_6.png) Change zoom of the report to fit the page width to the screen width.

![](../images/img_7.png) Change zoom of the report to display only one full page. More than one page by the width can be output.

![](../images/img_8.png) Set the report zoom.

* **Search Panel**


The search panel is used to search some text in the report. On the main toolbar this option can be enabled by clicking the binocular icon. All controls for search are placed on a single panel.


![](../images/topics/Viewer.Toolbar_3.png)


![](../images/img_1.png) Close the search panel.

![](../images/img_2.png) The field to put a text that should be found.

![](../images/img_3.png) The button Find Previous to run search.

![](../images/img_4.png) The button Find Next to run search.

![](../images/img_5.png) If the flag is set, then search will be repeated considering the case.

![](../images/img_6.png) If the flag is set, then search will be done considering the whole word.
