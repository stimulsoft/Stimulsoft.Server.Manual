## Formats with Fixed Page Layout


**Stimulsoft Reports.Server** supports three exports with fixed page layout. This means that all elements of a page can be placed at any part of a page. In this case, if to change a position of one element then other components position will not be changed. These are formats to [PDF](PDF/index.md) (Portable Document Format), [Microsoft Power Point File](Microsoft_Power_Point_File.md) and [XPS](XPS.md) (XML Paper Specification).
