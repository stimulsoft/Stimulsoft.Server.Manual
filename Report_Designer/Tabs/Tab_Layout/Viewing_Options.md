## Viewing Options


The group contains items to configure various settings of displaying some staff on the page in the report template.


![](../../../images/topics/Report_Designer.Tabs.Tab_Layout.Viewing_Options_1.png)


![](../../../images/img_1.png) The command **Show Grid**. Makes the grid on the page visible in the report template.

![](../../../images/img_2.png) The command **Align to Grid**. Automatically aligns by the grid the components on the page.

![](../../../images/img_3.png) The command **Grid Mode**. Sets the grid style. The grid may look as lines or dots.

![](../../../images/img_4.png) The command **Show Headers**. This command provides the ability to show or hide headers of bands.

![](../../../images/img_5.png) The command **Show Order**. Shows or hides the order of placing items on the page of a report template.

![](../../../images/img_6.png) The command **Show Ruler**. Shows rulers on the page.

![](../../../images/img_7.png) The button to call the menu [Parameters](Options.md).
