## Status Bar


The status bar is placed under the designer window. The picture below shows a status bar of the Standard UI.


![](../../images/topics/Report_Designer.Tabs.Status_Bar_1.png)


The bar contains 4 sections:

![](../../images/img_1.png) **Units**. This field shows current units in a report. It is possible to change them.

![](../../images/img_2.png) Start Report Checker. The Report Checker will analyze the report, resulting in an **errors**, **warnings**, **information messages**, **report rendering messages** in this report:


![](../../images/topics/Report_Designer.Tabs.Status_Bar_2.png)


![](../../images/img_3.png) Name of Selected Component.

![](../../images/img_4.png) Shows cursor coordinates on a page of a report template. (Х:0,0 ; Y:0,0) coordinates corresponds to the top left corner of a page of a report template.

![](../../images/img_5.png) The slider to zoom reports.
