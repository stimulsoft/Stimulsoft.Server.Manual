## Borders


This group contains the commands to setup border components.


![](../../../images/topics/Report_Designer.Tabs.Tab_Design.Borders_1.png)


![](../../../images/img_1.png) Sets or removes borders from all sides of a component.

![](../../../images/img_2.png) Sets or removes borders from each side of a component.

![](../../../images/img_3.png) Sets a border color of a component.

![](../../../images/img_4.png) Sets the shadow of a component.

![](../../../images/img_5.png) Sets a background color of a component.

![](../../../images/img_6.png) Sets a type of the border line.

![](../../../images/img_7.png) The button to call the border editor.


![](../../../images/topics/Report_Designer.Tabs.Tab_Design.Borders_2.png)


![](../../../images/img_1.png) Using the visual editor you can set the borders.

![](../../../images/img_2.png) In this field you can change the style of the border (e.g., solid, point, double, etc.).

![](../../../images/img_3.png) In this field you can set shadow options.

* The **Drop Shadow** parameter shows/hides the shadow of the border.

* Here you can change the size of the shadow.

* Here you can change the shadow color.
