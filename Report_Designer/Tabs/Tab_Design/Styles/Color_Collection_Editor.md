## Color Collection Editor


**Color Collection Editor** provides an opportunity to create a certain collection of colors for the style:


![](../../../../images/topics/Report_Designer.Tabs.Tab_Design.Styles.Color_Collection_Editor_1.png)


![](../../../../images/img_1.png) The button **Add**. When you click it the collection of colors will be added. By default, the color **White** is added. Then, press 
![](../../../../images/topics/Report_Designer.Tabs.Tab_Design.Styles.Color_Collection_Editor_2.png)
and select the desired color.

![](../../../../images/img_2.png) The button **Remove**. When you click it the selected color is removed from the collection of colors

![](../../../../images/img_3.png) **Up** and **down** arrows are used to move the selected color in the list.

![](../../../../images/img_4.png) The list of color collections.
