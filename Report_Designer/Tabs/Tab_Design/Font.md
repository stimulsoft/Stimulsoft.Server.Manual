## Font


This group is used to output text with the specified font type, color etc.


![](../../../images/topics/Report_Designer.Tabs.Tab_Design.Font_1.png)


![](../../../images/img_1.png) Select the font type of the text components on the current page.

![](../../../images/img_2.png) Select font size of the text components on the current page.

![](../../../images/img_3.png) Sets the font style as Bold, Italic, Underlined.

![](../../../images/img_4.png) Set the font color of the text components on the current page.
