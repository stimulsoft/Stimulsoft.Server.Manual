## Tab Styles


You can completely change the design of charts, ranging from basic colors and ending with shadows, borders, and so on. You can do this in the tab **Styles**.


![](../../images/topics/Report_Designer.Charts_Editor.Tab_Styles_1.png)


![](../../images/img_1.png) The button is used to call the style designer. In the designer, you can create a style for the chart and the collection of styles for other components.

![](../../images/img_2.png) In this panel you can see the list of styles that are available by default.

![](../../images/img_3.png) The panel **Preview**. This panel displays the chart and immediately previews changes made in real time.

![](../../images/img_4.png) The button **More Styles**. When you click it you will see the list of styles available by default.


> **Information**
>
> If the **AllowApplyStyle** is enabled then the style will be applied. If you disable the **AllowApplyStyle** then the parameters of series will be considered.
