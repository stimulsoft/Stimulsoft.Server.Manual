## Styles Tab


The final stage of the creating a cross-tab is to define its style.


![](../../images/topics/Report_Designer.Cross-Tab_Editor.Styles_1.png)


![](../../images/img_1.png) The button calls the style designer.

![](../../images/img_2.png) The list of preset styles available for cross-table. If you need a different style, you should call a designer to create a new style. In order to select the desired style, simply select it. At the same time the preview panel will display the structure of a cross-tab with the applied style.

![](../../images/img_3.png) The preview panel to see the structure a cross-tab. A red box around the cell indicates that the cell is selected.
