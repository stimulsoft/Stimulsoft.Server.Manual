## Tab Tools


On this tab you can find the following elements.


![](../images/topics/Tab_Tools_1.png)


![](../images/img_1.png) The **Share** command is used to call the sharing menu;

![](../images/img_2.png) The **Download** button saves the item as a file of a certain type. After selecting this command the dialog box will appear. In the dialog you should determine a saving location and click Save. You should know that not every item has this command.

![](../images/img_3.png) The **Versions** button calls the appropriate menu.

![](../images/img_4.png) The **Using** button calls the appropriate menu.

![](../images/img_5.png) The **Log** button calls the appropriate menu.

![](../images/img_6.png) When creating an item the unique key of the item is automatically generated. In order to obtain this key, select the command. The unique key of the item is needed for future access to this item, when using API of the report server in third-party applications. After selecting this command, the key will be displayed on the Access Key panel.
