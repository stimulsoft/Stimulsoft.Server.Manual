## Menu Create

The tab **Create** contains commands with which new items are added to the list of items.


![](../../images/create.png)


![](../../images/img_1.png) The command [Report](Report.md) calls a menu to create a report or upload it from the file.

![](../../images/img_2.png) The command [Dashboard](Dashboard.md) calls a menu to create a dashboard or upload it from the file.

![](../../images/img_3.png) The command [Data Source](Data_Source/index.md) is used to create a new data source

![](../../images/img_4.png) The command [Scheduler](Scheduler/index.md) is used to execute commands on the schedule, add new ones.

![](../../images/img_5.png) The command [Folder](Folder.md) is used to organize and store items in right places. You can create a hierarchy of folders in the list of items.

![](../../images/img_6.png) The command [Calendar](Calendar.md) is used to create a calendar. For example, you can create a calendar to run the scheduler.

![](../../images/img_7.png) The command [File](File.md) is used to add a file to the list of items.

![](../../images/img_8.png) The command [File Storage](File_Storage.md) is used to create directory in which can save the contents of the server items.

![](../../images/img_9.png) The command [Contact List](Contact_List.md) is used to create a list of contacts. For example, the contact list may be used by the scheduler, i.e. can form a permanent list of contacts by what users will receive the results by e-mail.

![](../../images/img_10.png) The command [Mailing](Mailing.md) calls a menu to create a mailing list.

![](../../images/img_11.png) Command to call the creation of a [Link](Link.md).
